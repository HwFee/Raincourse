# GUI 使用说明

本文件聚焦图形界面使用流程。项目完整说明请查看 README.md。

## 功能

- 独立窗口运行（非终端交互）
- 支持扫码登录与本地会话
- AI 辅助答题与实时日志
- 题目导出与 API 配置
- 支持打包为 EXE

## 快速开始

1. 安装依赖

```bash
pip install -r requirements.txt
```

2. 启动 GUI（推荐）

```bat
run_gui.bat
```

或

```bash
pythonw gui.py
```

3. 调试模式（显示控制台）

```bash
python gui.py
```

## 基本流程

1. 选择登录方式并完成登录。
2. 进入 AI 页面，选择课程与作业。
3. 点击开始答题，查看右侧实时日志。
4. 完成后手动核对答案并提交。

## 打包 EXE

```bat
build.bat
```

打包完成后可在 dist 目录查看可执行文件。

## 常见问题

- 启动闪退：先使用 python gui.py 查看报错信息。
- 无法调用模型：检查设置中的 API 密钥与网络连通性。
- 导出失败：确认导出目录有写入权限。
