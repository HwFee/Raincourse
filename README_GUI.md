# 雨课堂AI答题助手 - GUI版本

## 🎯 功能特点

- ✅ 独立窗口应用程序（不是终端）
- ✅ 美观的图形界面
- ✅ 支持账号登录和会话保存
- ✅ AI智能答题
- ✅ 实时答题日志
- ✅ 可打包成EXE文件

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行GUI版本（无终端黑窗）

推荐双击：

```bash
run_gui.bat
```

或在命令行执行：

```bash
pythonw gui.py
```

仅用于调试时再使用：

```bash
python gui.py
```

### 3. 打包成EXE

双击运行 `build.bat`，会自动：
- 安装 `requirements.txt` 中所有依赖
- 清理旧文件
- 打包成EXE
- 校验产物是否生成
- 输出到 `dist` 目录（`RaincourseAIHelper.exe`）

## 🔧 技术栈

- **后端**: Python + PyWebView
- **前端**: HTML + CSS + JavaScript
- **打包**: PyInstaller

## 📝 使用说明

1. 启动程序后，选择登录方式
2. 登录成功后，选择要答题的考试
3. 点击"开始答题"，AI会自动答题
4. 查看实时日志了解答题进度

## ⚠️ 注意事项

- 首次运行需要安装依赖
- 打包后的EXE文件较大（约50MB）
- 需要网络连接才能使用AI功能
